# Crypto Sentiment Analysis

## Objective

This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid historical trading data.

## Datasets

1. Bitcoin Fear & Greed Index
2. Hyperliquid Historical Trader Data

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Findings

- Extreme Greed had the highest average profit per trade (67.89).
- Fear periods generated the highest total profit (3.36M).
- Extreme Greed showed the highest win rate (46.49%).
- Market sentiment has a measurable impact on trading outcomes.

## How to Run

```bash
pip install -r requirements.txt
jupyter notebook
```

Open Analysis.ipynb and run all cells.
